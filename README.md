# 饮用水订购小程序

一款面向校园场景的饮用水在线订购管理系统，支持老师下单和管理员订单处理。

## 项目简介

本小程序采用微信小程序原生开发，为学校饮用水配送场景提供完整的解决方案。系统分为客户下单端和管理后台两端，客户可通过小程序快速下单，管理员可实时查看订单状态并处理配送。

## 功能特性

### 下单页面（客户/老师端）
- 📝 **信息录入**：姓名、联系方式、所属学院
- 🛒 **产品选择**：桶装水/箱装水，支持数量调整
- 💳 **付款方式**：现结/挂账两种选择
- 🚚 **配送信息**：配送时间、送达地址
- 📋 **备注功能**：支持特殊需求说明
- ✅ **表单验证**：完整的手机号格式验证，确保信息准确

### 管理页面（管理员端）
- 📊 **实时统计**：今日订单总数、待配送、已完成、总数量
- 📅 **历史查询**：支持按日期查询历史订单数据
- 🔍 **订单筛选**：全部/待配送/已完成快速切换
- ✏️ **订单处理**：一键标记配送、删除订单
- 📞 **快捷联系**：点击即可拨打客户电话
- 🔔 **新单提醒**：每30秒自动检测新订单，页面标题闪烁提醒
- 📤 **数据分享**：支持分享给其他管理员

### 技术亮点
- 使用微信本地存储实现数据持久化
- 自动跨日期订单归档
- 响应式布局设计
- 完整的错误处理机制
- 优雅的用户交互反馈

## 技术栈

- **框架**：微信小程序原生框架
- **语言**：JavaScript (ES6+)
- **样式**：WXSS + CSS3 弹性盒模型
- **数据**：wx.setStorageSync 本地存储
- **组件**：原生组件 + 自定义样式

## 快速开始

### 环境要求
- 微信开发者工具最新版本
- 微信小程序AppID（测试号或正式号均可）

### 运行步骤
1. 打开微信开发者工具
2. 选择「导入项目」
3. 选择本项目目录
4. 点击「导入」完成配置
5. 点击「编译」运行查看效果

### 真机测试
- 使用「预览」生成二维码，手机扫码体验
- 使用「真机调试」连接真机进行调试

## 数据存储说明

本小程序使用微信本地存储功能：
- `orders`：今日订单列表
- `orderHistory`：历史订单归档（按日期存储）
- `lastOrderDate`：上次访问日期

每日0点自动将当日订单归档至历史记录，确保数据不丢失。
